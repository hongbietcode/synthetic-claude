# Content Creation

Content creation tools for prompting, documentation, and LLM applications.

## Installation

```bash
/plugin install synthetic-claude@content-creation
```

## Components

### Commands (5)

| Command | Description |
|---------|-------------|
| `/create-project-memory-skills` | Copy memory skills to project |
| `/current-prompt-create` | Create current_prompt.md file |
| `/ecp` | Execute prompt from file |
| `/notebook-edit` | Suggest Jupyter notebook changes |
| `/py2notebook` | Convert Python to notebook |

### Skills (7)

| Skill | Description |
|-------|-------------|
| **prompting** | Model-agnostic LLM prompt best practices |
| **doc-coauthoring** | Collaborative documentation workflows |
| **internal-comms** | Internal communication templates |
| **llm-apps-creator** | LangChain application patterns |
| **power-agent-creator** | Build Claude Code-like agents |
| **web-artifacts-builder** | React + Tailwind + shadcn/ui bundling |
| **templates** | Project memory skill templates |
