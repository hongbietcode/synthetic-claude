# Design Studio

Design tools for UI/UX, frontend design, and visual assets.

## Installation

```bash
/plugin install synthetic-claude@design-studio
```

## Components

### Agents (1)

| Agent | Purpose |
|-------|---------|
| ui-ux-designer | Expert design guidance with WCAG accessibility focus |

### Commands (1)

| Command | Description |
|---------|-------------|
| `/design-guide` | Design analysis without modifying code |

### Skills (6)

| Skill | Description |
|-------|-------------|
| **frontend-design** | Production-grade interfaces with bold aesthetic |
| **canvas-design** | Visual art creation (PDF/PNG) via design philosophy |
| **algorithmic-art** | p5.js generative art with seeded randomness |
| **theme-factory** | 10 curated professional themes |
| **brand-guidelines** | Anthropic brand styling guide |
| **slack-gif-creator** | Optimized GIF creation (128x128, 480x480) |

## Design Philosophy

- **Philosophy-first**: Define aesthetic worldview before execution
- **Accessibility**: WCAG compliance in all designs
- **Reproducibility**: Seeded randomness for consistent outputs
