# Document Suite

Document processing skills for PDF, DOCX, PPTX, and XLSX files.

## Installation

```bash
/plugin install synthetic-claude@document-suite
```

## Components

### Skills (4)

| Skill | Description |
|-------|-------------|
| **pdf** | Extract, analyze, transform PDF content, form filling, OCR |
| **docx** | Word document creation, modification, tracked changes |
| **pptx** | PowerPoint presentation generation and editing |
| **xlsx** | Excel spreadsheet creation with formulas |

## Features

### PDF Processing
- Text and table extraction
- PDF merging and splitting
- Form field detection and filling
- Bounding box analysis
- Image conversion

### DOCX Processing
- Document creation from scratch
- Tracked changes and comments
- Redlining workflow support
- OOXML manipulation

### PPTX Processing
- Presentation creation (scratch, template, existing)
- Slide inventory and rearrangement
- Thumbnail generation
- HTML to PPTX conversion

### XLSX Processing
- Spreadsheet creation with multiple sheets
- Formula support with mandatory recalculation
- Data manipulation and formatting
